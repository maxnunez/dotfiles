┌─ CRÍTICAS ──────────────────────────────────────────────────┐
│                                                              │
│ [1] Git config usa nano                                      │
│     ~/.gitconfig tiene: editor = nano                        │
│     Debería ser: editor = nvim                               │
│     (EDITOR=nvim en .zshrc, pero git tiene su propio config) │
│     Para corregir: git config --global core.editor nvim      │
│                                                              │
│ [2] FZF_ALT_COMMAND tiene typo                               │
│     En ~/.zshrc: --exlude .git (falta la 'c')               │
│     Debería ser: --exclude .git                              │
│                                                              │
